FUNCTION

fn_search_students

Tìm kiếm học viên full-text theo username, full_name hoặc school_name. Hàm trả về SET OF RECORD, lọc sẵn is_deleted = FALSE.

FUNCTION

fn_search_courses_advanced

Tìm kiếm nâng cao kết hợp lọc theo keyword, danh mục và trạng thái. Hàm JOIN 4 bảng, tổng hợp total_students và avg_progress.

PROCEDURE

sp_enroll_student

Gọi procedure đăng ký học viên vào khóa học bằng CALL. Hệ thống sẽ kích hoạt Trigger trg_touch_student_activity_on_enroll → tự động cập nhật student_streaks.

🔔 TRIGGER: trg_touch_student_activity_on_enroll → student_streaks
PROCEDURE

sp_update_course_progress

Cập nhật tiến độ học. Nếu đạt 100%, Trigger trg_notify_course_completion sẽ tự động INSERT thông báo hoàn thành vào notification_users.

🔔 TRIGGER: trg_notify_course_completion → notification_users
TRANSACTION

Transaction End-to-End

Kịch bản dài nhất: BEGIN → Validate → CALL sp_update_course_progress → INSERT comment → kiểm tra trigger side-effects → COMMIT. Một trong hai thất bại thì toàn bộ ROLLBACK.

BEGIN → CALL sp_update_course_progress → INSERT comments → COMMIT
TRANSACTION

Wallet Transfer To Admin

Demo chuyển tiền trực quan: BEGIN → khóa ví nguồn/đích (FOR UPDATE) → kiểm tra status=active + số dư → cập nhật số dư → ghi transaction_logs + transaction_action_logs → COMMIT/ROLLBACK logic.

BEGIN → CHECK SOURCE → CHECK ADMIN → UPDATE wallets → INSERT logs → COMMIT/ROLLBACK
PROCEDURE

sp_soft_delete_user

Xóa mềm tài khoản bằng cách set is_deleted = TRUE — không xóa khỏi DB vật lý. Trigger trg_auto_update_users_timestamp tự cập nhật updated_at.

🔔 TRIGGER: trg_auto_update_users_timestamp → users.updated_at
PROCEDURE

sp_soft_delete_course

Xóa mềm khóa học bằng cách set is_deleted = TRUE. Trigger trg_auto_update_courses_timestamp tự cập nhật updated_at.

🔔 TRIGGER: trg_auto_update_courses_timestamp → general_courses.updated_at
